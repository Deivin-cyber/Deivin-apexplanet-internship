# Deivin-apexplanet-internship

ApexPlanet Cybersecurity & Ethical Hacking Internship

This repository includes all the tasks, projects, and work completed during the Cybersecurity & Ethical Hacking Internship at ApexPlanet Software Pvt. Ltd.
It contains the learning materials and practical tasks done throughout the internship.

Task 1: Foundations of Cybersecurity

Introduction to the basic concepts of cybersecurity and information security.

Task 2: Network Security & Scanning

Learning about network security and using tools to scan and analyze networks.

Task 3: Web Application Security

Understanding common vulnerabilities in web applications and how to identify them.

Task 4: Exploitation & System Security

Learning about system vulnerabilities and basic exploitation techniques to understand how attacks happen.

Task 5: Capstone Project & Incident Response

A final project that applies the learned skills and focuses on handling security incidents and responses.
