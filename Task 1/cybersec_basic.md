CIA Triad

The CIA Triad is a basic security model that explains the three main goals of protecting information.

Confidentiality:
Confidentiality means keeping information private so that only authorized people can access it. For example, using a strong password to protect your email.

Integrity:
Integrity means the data should be correct and should not be changed or damaged. For example, a bank must keep the correct record of your account balance.

Availability:
Availability means authorized users should be able to access systems or information whenever they need them. For example, a website should always be working and accessible.

Threat Types

Threats are different kinds of attacks that can harm computers or steal information.

Phishing:
Phishing is when attackers send fake emails or create fake websites to trick people into giving sensitive information like passwords or credit card details.

Malware:
Malware is harmful software created to damage or control a computer system. Examples include viruses, worms, and Trojans.

DDoS (Distributed Denial-of-Service):
A DDoS attack sends a huge amount of traffic to a server so that it becomes overloaded and stops working for normal users.

SQL Injection:
SQL injection is an attack where hackers exploit a website’s database to access, change, or delete stored data.

Brute Force:
Brute force is a method where attackers try many possible password combinations until the correct one is found.

Ransomware:
Ransomware is a type of malware that locks a user's files and asks for money to unlock them.

Attack Vectors

Attack vectors are the ways attackers use to enter or damage a system.

Social Engineering:
Social engineering is when attackers trick people into revealing confidential information by using psychological manipulation.

Wireless Attacks:
Wireless attacks target Wi-Fi networks to gain unauthorized access. This may involve breaking the Wi-Fi password or creating a fake Wi-Fi network.

Insider Threats:
An insider threat happens when a person who already has access to a system, such as an employee, misuses that access to steal data or damage the system.
