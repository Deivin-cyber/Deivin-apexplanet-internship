File System Navigation

pwd:
Print Working Directory. It shows the current folder or location you are working in.

ls:
List directory contents. It displays the files and folders inside the current directory.

cd:
Change Directory. It is used to move from one folder to another.

File & Directory Permissions

chmod:
Change Mode. It is used to change the permission of a file or directory.

chown:
Change Owner. It is used to change the owner of a file or folder.

Package Management

apt:
A package management tool used to install, update, and remove software packages.

dpkg:
A tool used to install and manage Debian package files directly.

Networking Commands

ifconfig:
Shows the configuration details of network interfaces.

ping:
Used to check if a system or website is reachable over the network.

netstat:
Displays information about active network connections and routing tables.

traceroute:
Shows the route or path that data packets take to reach a destination.
